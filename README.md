# alx-higher_level_programming
This phase focuses on high level programming languages such as python, SQL, JavaScript and so on. 

Python has efficient high-level data structures and a simple but effective approach to object-oriented programming. 

The first full web application: the AirBnB clone

Created Airbnb clone using python.

Wrote a command interpreter to manage AirBnB objects. 

The Airbnb clone project was grouped into segments in the following order;

0x00. AirBnB clone - The console

0x01. AirBnB clone - Web static

0x02. AirBnB clone - MySQL

0x03. AirBnB clone - Deploy static

0x04. AirBnB clone - Web framework

0x05. AirBnB clone - RESTful API

0x06. AirBnB clone - Web dynamic

At the end of the project an Airbnb clone web app was created.
